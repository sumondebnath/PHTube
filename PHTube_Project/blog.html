1. Discuss the scope of var, let, and const

    • var

      var is a keyword, it works functional scope and global scope. it also declear variables, re-declear variables and also assign value in the same variables.

    • let

      let is a keyword, it is a block-scope variable. it is one-time declearation variable, multiple time assign value but no re-declearation with same name.

    • const

      const is a keyword, it is also block-scope variable. it is one-time declearation variable, no assign value and no re-declearation same name.
